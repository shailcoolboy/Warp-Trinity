Requirements

All the software requirements for this tutorial are described here.

Software Requirements

This tutorial does not require any hardware. The design is validated in simulation using the tools.

The following tools must be installed on your computer:
- MATLAB 2007a or later
- Xilinx ISE 10.1.02 or later
- Xilinx System Generator 10.1.2 or later

Setup

Download and install the WARP Repository to your MATLAB installation. The instructions are available at http://warp.rice.edu/trac.
